# ⛪ Church Website

A modern and responsive church website designed to provide members and visitors with easy access to church information, events, and resources.

---

## 📌 Overview

This project is a fully responsive frontend website built to enhance a church's online presence. It allows users to easily access important information such as service times, events, and announcements.

---

## ✨ Features

- 📱 Responsive design (mobile, tablet, desktop)
- 🏠 Homepage with welcome message
- 📅 Events and announcements section
- 📰 Weekly bulletin display
- 📍 Contact and location information
- 🎨 Clean and user-friendly UI

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript  

---

## 🎯 Purpose

The goal of this project is to create a simple, accessible, and engaging platform for church members and visitors to stay informed and connected.
